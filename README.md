# BuildWithDev

BuildWithDev is a public product lab where experiments, SaaS prototypes, and internet tools are built in public.

Each product includes:
- UI system
- prompts used
- architecture notes
- public GitHub repository
- screen recording demo

## Project Structure
- `/projects`: Product showcase and build logs
- `/research`: Product research, growth experiments, and theory
- `/content`: MDX files for products and research
- `/components`: Poster Modernist UI system

## Deployment & Domain
This project is deployed on Vercel with continuous deployment from the `main` branch.

### To connect buildwithdev.xyz:
1. Open Vercel dashboard.
2. Select the `buildwithdev` project.
3. Go to **Settings → Domains**.
4. Add `buildwithdev.xyz`.
5. Follow the DNS instructions provided by Vercel.
